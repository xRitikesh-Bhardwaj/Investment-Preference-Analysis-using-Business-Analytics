# 📊 Investment Preference Analysis using Business Analytics

## 📌 Overview

This project analyzes investment preferences using Business Analytics techniques. It explores investor behavior through data visualization, descriptive statistics, and correlation analysis to generate meaningful business insights.

## 🎯 Objectives

- Analyze investment preferences
- Study gender distribution
- Perform descriptive statistics
- Explore savings objectives
- Analyze investment duration
- Identify information sources
- Understand investment expectations
- Perform correlation analysis

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab / Jupyter Notebook

## 📈 Key Analysis

- Data Overview
- Gender Distribution
- Descriptive Statistics
- Investment Preference Analysis
- Factors Affecting Investment Decisions
- Savings Objectives
- Information Sources
- Investment Duration
- Investment Expectations
- Correlation Analysis
